# С 8 Марта! 💐

Поздравительный сайт для одногруппниц.

## Как запустить

### Вариант 1: Локальный сервер (рекомендуется)

В папке проекта выполни:

```bash
npm run serve
```

Открой в браузере: **http://localhost:3000**

(Если `serve` не установлен, он установится автоматически через `npx`.)

### Вариант 2: Без установки

```bash
npx serve . -l 3000
```

Затем открой **http://localhost:3000**.

### Вариант 3: Только сборка и открытие файла

Если нужна только сборка TypeScript:

```bash
npm run build
```

После этого можно открыть `index.html` двойным кликом (модули в некоторых браузерах по `file://` могут не загрузиться — лучше использовать вариант 1 или 2).

---

**Структура:** `index.html` — главная страница, `styles.css` — стили, `dist/main.js` и `dist/data.js` — скрипты (собираются из `src/*.ts` командой `npm run build`).
